                        DEPLOYMENT INSTRUCTIONS
================================================================================

QUICK START (Follow in order):

1. PREPARE SERVER
   - Provision server (AWS, DigitalOcean, etc.)
   - Install OS dependencies (PHP, MySQL, Node.js, Nginx)
   - Clone repository

2. SETUP BACKEND
   cd admin
   composer install --optimize-autoloader --no-dev
   php artisan key:generate --env=production
   php artisan migrate --force
   php artisan db:seed

3. SETUP FRONTEND
   npm install
   npm run build

4. CONFIGURE SERVER
   - Setup Nginx configuration
   - Install SSL certificate
   - Set file permissions
   - Configure PHP-FPM

5. OPTIMIZE & SECURE
   php artisan config:cache
   php artisan route:cache
   php artisan optimize
   Set APP_DEBUG=false

6. VERIFY & MONITOR
   - Test API endpoints
   - Test admin panel login
   - Check error logs
   - Monitor application

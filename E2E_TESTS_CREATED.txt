                        QUICK START COMMANDS

1. INSTALL DEPENDENCIES
   $ cd admin
   $ npm install

2. INSTALL PLAYWRIGHT BROWSERS (optional, auto-installed on first run)

6. SPECIFIC TEST
   $ npx playwright test -g "Login with valid admin"

7. VIEW REPORT
   $ npx playwright show-report
